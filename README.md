#  :books: js-mini-project

**1. colorFlipper**<br/>

**2. counter**<br/>

**3. carousel**<br/>

-> 객체, document.querySelector(), addEventListener(), textContent, array.length, Math.floor(), Math.random()
<br/><br/>

**4. responsiveNavigation**<br/>

**5. responsiveSidebar** <br/>

<br/><br/>

<br/><br/>

**10. scrollNavigation**<br/>
